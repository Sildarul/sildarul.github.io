# La Boîte à Vider - Sildarul

Page GitHub Pages prête à l'emploi.

## Installation GitHub Pages

1. Mets `index.html` et le dossier `assets` à la racine du dépôt.
2. Va dans `Settings > Pages`.
3. Source : branche `main`, dossier `/root`.
4. Attends le déploiement.
5. Mets l'URL GitHub Pages dans ta description YouTube.

## Structure

- `index.html` : page principale.
- `assets/` : toutes les images nécessaires.
- `assets/pack_outbreak_tout_en_un.svg` : visuel inventé du pack, pas une miniature YouTube.
- `README.md` : ce fichier.
